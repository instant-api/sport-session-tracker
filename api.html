Instant Sport Session Tracker API

GET /

Return this page

GET /public/*

Everything under the /public url is statically served.

GET /workouts

Return an object with a list of workout object and the total { result: [], total: 123 }

Accept the following query parameters:

  • offset: Number of items to skip
  • limit: Number of items to return (default 10)
  • places: A list a places slugs
  • users: A list of user's username
  • order: 'asc' or 'desc'
  • sort: One of 'date' 'place' 'user' 'distance' 'duration' 'speed'

GET /places

Return an object with a list of places object and the total { result: [], total: 123 }

Accept the following query parameters:

  • offset: Number of items to skip
  • limit: Number of items to return (default 10)

GET /place/:slug

Return a place object

POST /action/signup

Signup to create a new account

  • JSON Body { username, password, firstName, lastName, age, weight, height } (all properties are required)
  • Return an object with the token of the created user: { token: "" }

POST /action/login

Login using username and password

  • JSON Body { username: "", password: "" } (all properties are required)
  • Return an object with the token of the user: { token: "" }

Authenticated routes

The following routes require a Require an Authorization header of Bearer TOKEN

GET /me

Return the me object: { username, token, firstName, lastName }

POST /action/create-workout

Create a new workout

  • JSON Body: { date, duration, distance, place } (date must be an ISO Date string)
  • Return an object with { id }

POST /action/create-place

Create a new place

  • JSON Body: { name, slug, lng, lat }
  • Return an object with { slug }
